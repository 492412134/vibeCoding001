配置示例

📝 场景：验证身份证是否为残疾人

API地址：https://disability.gov.cn/api/verify

请求体模板：

{
  "idCard": "${idCard}",
  "requestTime": "${currentTime}"
}

响应字段路径：data.isDisabled

成功条件：response == true

说明：调用接口后，如果data.isDisabled为true，则规则通过
